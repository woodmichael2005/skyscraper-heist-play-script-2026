## What this project is

Skyscraper Heist is a web-based game script and playable board game experience that reworks Snakes & Ladders into a cyberpunk setting. Instead of the familiar snakes and ladders, the board uses guards and secret vents, creating a progression style that matches the neon city theme.

The project is built with HTML, CSS, and JavaScript, and it emphasizes responsive turn flow, clear feedback, and a polished visual layer. A hacker-terminal style activity log and special power tiles help keep each match varied while leaving the core rules simple to understand.

## Script Features

- Cyberpunk Snakes & Ladders board design
- Guards used as the hazard path element
- Secret vents used as the upward movement path
- Special power tiles for added match variety
- Two-player mode for head-to-head play
- Animated dice rolling for turn visibility
- Smooth token movement across the board
- Neon UI styling with a hacker terminal log
